# CPI Inflation Analysis using Excel

## Project Overview
This project analyzes Consumer Price Index (CPI) data using Microsoft Excel.

## Objectives
- Clean and prepare CPI data
- Handle missing values through imputation
- Analyze inflation trends
- Compare inflation behavior before and after COVID-19
- Create charts and visual insights

## Tools Used
- Microsoft Excel
- Charts & Visualization
- Data Cleaning
- Statistical Analysis

## Project Structure
- Executive Dashboard
- Data Cleaning & Imputation
- Trend Analysis
- Category Comparison
- Pre/Post COVID Analysis
- Monthly Analysis

## Key Skills Demonstrated
Excel, Data Analysis, Business Analytics, Data Visualization, Reporting
